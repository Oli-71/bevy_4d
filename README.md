This is a demo to visualize a 4D scene from different viewpoints and with different transformations,
to give an intuition of how a 4D world would look and how it relates to our 3D world ("spaceland") and a 2D world ("flatland") experience.
 
The content of a 2D/3D/4D scene is incrementally added to the screen. 
The user can explore the different aspects step by step with increasing irritation ;-) 

It's implemented in Bevy, a Rust-based gaming engine.
